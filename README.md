# Knights Challenge

O desafio Knights visa a construção um sistema cadastro de heróis utilizando as
seguintes tecnologias:



## Stack utilizada

**Front-end:** • Vue.js

**Back-end:**
• NodeJS (Express.js | NestJS)
• MongoDB | Redis




## Rodando localmente

Instalar as dependências

```bash
  yarn
```

Inicializar Redis.

```bash
  docker-compose up
```

Inicializar o projeto.

```bash
  yarn start:dev
```
